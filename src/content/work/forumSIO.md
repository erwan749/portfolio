**Contexte :**  
Premier projet réalisé pendant mon BTS SIO. L’objectif était de créer un **forum web fonctionnel** permettant aux utilisateurs de créer des comptes, poster des messages et gérer des catégories de discussion, avec une **première couche de sécurité** pour éviter les injections SQL basiques, et un peu de **JavaScript** pour améliorer l’expérience utilisateur (popups pour suppression, connexion et inscription).

**Technologies utilisées :**  
- PHP brut pour la logique côté serveur  
- SQL pour la gestion des données  
- HTML/CSS pour l’interface  
- JavaScript pour certaines interactions (popups, confirmation de suppression, formulaire de connexion/inscription)  

**Description :**  
- Création d’un système d’inscription et d’authentification des utilisateurs.  
- Gestion des messages et des catégories dans une **base SQL**.  
- Validation des formulaires et **sécurisation des entrées pour éviter les injections SQL basiques**.  
- Affichage dynamique des discussions et des messages postés.  
- **Popups interactifs en JavaScript** pour améliorer l’UX sur certaines actions.

**Ce que j’ai appris :**  
- Développement web **à la main** sans framework  
- Gestion des bases de données relationnelles avec SQL  
- Premières notions de **sécurité web** contre les injections SQL  
- Utilisation de **JavaScript léger** pour améliorer l’interaction  
- Structuration d’un projet web complet côté serveur et front-end
