**Technologies utilisées :**  
- Full .NET (API + interface)  
- WebSocket (WSS) pour la communication en temps réel entre l’API et les sessions  
- Gestion des données en SQL  
- Versioning avec Git

**Description :**  
- Jeu de gestion **tour par tour**, chaque tour représentant une période de travail.  
- L’utilisateur doit **engager des développeurs**, attribuer des projets et gérer les ressources.  
- L’API .NET gère la logique de jeu, les règles et l’état des équipes.  
- **Communication en temps réel** via WSS pour synchroniser les sessions et les actions des joueurs.  
- Interface en TerminalUI pour interagir facilement avec le jeu.

**Ce que j’ai appris :**  
- Développement d’une application Full .NET complète  
- Conception et structuration d’une API pour gérer un jeu complexe  
- Mise en place de **WebSocket** pour communication temps réel  
- Gestion des états de jeu et des tours via une base SQL  
- Collaboration avec Git pour versionner et suivre le projet
