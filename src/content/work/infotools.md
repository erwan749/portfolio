Projet réalisé pendant ma deuxième année de BTS SIO (2024-2025). Infortool est une application **multiplateforme** pour la gestion des employés, des clients et des factures. Mon travail s'est concentré sur :  

- Le **CRM web** en Laravel  
- La **partie mobile MAUI**  
- L’**API Laravel** pour la communication entre la base de données et les applications externes  

La partie desktop C# a été développée par d’autres membres de l’équipe.

**Technologies utilisées :**  
- Laravel pour l’API et le CRM web  
- MAUI pour la partie mobile  
- SQL pour la gestion des données  
- Git pour le versioning et le suivi du projet  

**Description :**  
- Développement du **CRM web** pour gérer les employés, les clients et les factures.  
- Conception et création d’une **API REST en Laravel** pour permettre la communication entre la base de données et les applications externes.  
- Développement de la **version mobile avec MAUI**, intégrée à l’API pour consulter et gérer les données depuis un smartphone.  
- Mise en place de routes sécurisées et gestion des permissions utilisateurs.  
- Collaboration en équipe pour intégrer la partie desktop C#.

**Ce que j’ai appris :**  
- Développement Laravel orienté **API et gestion de données**  
- Développement mobile avec **MAUI** et intégration avec une API existante  
- Gestion des bases de données relationnelles et optimisation des requêtes  
- Travail en équipe sur un projet **multiplateforme**
